# Linux Commands Reference

This directory contains Linux commands organized by category:

## Files

- **[file-operations.md](file-operations.md)** - Commands for managing files and directories (ls, cd, cp, mv, rm, mkdir, etc.)
- **[file-viewing.md](file-viewing.md)** - Commands for viewing file contents (cat, less, more, head, tail, etc.)
- **[text-processing.md](text-processing.md)** - Commands for text manipulation (grep, cut, sort, tee, wc)
- **[system-info.md](system-info.md)** - Commands for system information (uname, uptime, df, du, free, etc.)
- **[process-management.md](process-management.md)** - Commands for managing processes (top, htop, ps, fuser)
- **[help-documentation.md](help-documentation.md)** - Commands for getting help (man, --help)
- **[ssh-network.md](ssh-network.md)** - SSH and network-related commands
- **[utilities.md](utilities.md)** - General utility commands (sudo, echo, date)

## Quick Reference

Browse the individual files above based on what you need to accomplish.
