# Utility Commands - Done

## `sudo -i`

Root shell:

```md
sudo -i
```

---

## `sudo su -`

```md
sudo su -
```

Exit root:

```md
exit
```

---

## `echo`

```md
echo hello
```

Append to file:

```md
echo hello >> file.txt
```

Test:

```md
cat file.txt
```

---

## `date`

```md
date
```

Save timestamp:

```md
date >> file.txt
```
