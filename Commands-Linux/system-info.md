# System Information Commands - Done

## `uname`

Display system information

```md
uname
uname -a
uname -r
```

---

## `uptime`

Show system uptime

```md
uptime
```

---

## `who`

Show who is logged on

```md
who
who -b
```

---

## `whoami`

Print current user

```md
whoami
```

---

## `which`

Locate a command

```md
which python
which ls
```

---

## `id`

Display user and group IDs

```md
id
id username
```

---

## `df`

Disk free:

```md
df
df -h
```

---

## `du`

Directory size:

```md
du .
du -sh .
```

---

## `free`

Memory:

```md
free
free -h
```

---

## `vmstat`

Memory + CPU:

```md
vmstat
vmstat 1
```
