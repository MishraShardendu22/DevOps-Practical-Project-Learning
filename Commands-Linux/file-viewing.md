# File Viewing Commands - Done

## `cat`

Display file content

```md
cat file.txt
cat file1.txt file2.txt
```

---

## `less`

Scrollable viewer

```md
less file.txt
```

---

## `more`

```md
more file.txt
```

---

## `head`

```md
head file.txt
head -n 5 file.txt
```

---

## `tail`

```md
tail file.txt
tail -n 5 file.txt
```

Live follow:

```md
tail -f file.txt
```

---

## `zcat`

View compressed file:

```md
zcat file.txt.gz
```

Test:

```md
gzip file.txt
zcat file.txt.gz
```
